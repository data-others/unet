## Introduction

This repository provides segmentation models for mouse, rat, marmosets, rhesus monkeys, and human. These models are easily adaptable and retrainable for various imaging modalities, streamlining the segmentation process and reducing the need for manual annotation.

The models can be used by [DSI Studio](https://dsi-studio.labsolver.org) and [UNet Studio](https://unet-studio.labsolver.org).
